Go to your self icon in the upper right >> Preferences >> Themes
Select "Light" or "Dark".
Paste the following color hex strings into the "Colors, Custom Theme".

Column BG : bg : @bg@
Unknown : emphFg : @emphFg@
Active Item : emphBg : @emphBg@
Active Item Text : emphFg : @emphFg@
Hover Item : softViolet : @softViolet@
Text Color : fg : @fg@
Active Presence : vividViolet : @vividViolet@
Mention Badge : vividRed : @vividRed@
Top Nav BG : emphBg : @emphBg@
Top Nav Text : emphFg : @emphFg@

Or just paste the following text into any chat channel, and then tap "Switch sidebar theme".

@_name@: @bg@,@emphFg@,@emphBg@,@emphFg@,@softViolet@,@fg@,@vividViolet@,@vividRed@,@emphBg@,@emphFg@
